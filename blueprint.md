# Blueprint: 생생 정보통 (Sangsang Jeongbotong)

## 서비스 개요
"생생 정보통"은 에브라임 시드(Ephraim Seed)가 소유하고 운영하는 정보 아카이브입니다. 사용자들에게 생활에 유용한 정보, 건강 팁, 절약 방법 등을 제공하는 미니멀하고 현대적인 정보 공유 블로그입니다. 계절에 따라 변화하는 디자인 요소와 깨끗한 UI를 통해 최상의 사용자 경험을 제공하는 것을 목표로 합니다.

## 프로젝트 상세 내용
### 디자인 및 스타일
- **레이아웃:** 반응형 웹 디자인을 지원하며, 중앙 집중형 레이아웃을 사용합니다.
- **색상 팔레트:** 현대적이고 깨끗한 화이트 배경에 강조색(파란색, 초록색 등)을 사용합니다.
- **타이포그래피:** 'Noto Sans KR' 폰트를 사용하여 가독성을 높였습니다.
- **특수 효과:** 
  - `canvas`를 이용한 계절별 파티클 효과.
  - 카드형 UI에 깊이감 있는 그림자와 부드러운 호버 애니메이션.
  - 배경에 미세한 노이즈 텍스처를 추가하여 프리미엄한 질감 구현.
- **데이터 시각화:** 바 차트, 통계 카드, 영향도 그리드 등 복잡한 경제 데이터를 시각적으로 전달하기 위한 전용 컴포넌트 추가.
- **컴포넌트:**
  - 고정 헤더와 내비게이션 바.
  - 그리드 형태의 포스트 카드 목록.
  - 검색 바 (실시간 필터링 기능).

### 주요 기능
- **포스트 목록:** 최근 게시글을 홈 화면에 노출하고, 전체 글 페이지에서 모든 게시글을 확인 가능.
- **검색 기능:** 제목 및 태그를 기반으로 한 실시간 게시글 검색.
- **계절 감지:** 현재 날짜를 기반으로 계절을 감지하여 UI 테마 및 파티클 효과 변경.
- **반응형 디자인:** 모바일, 태블릿, 데스크탑 환경에 최적화된 레이아웃.

1. **Kakao AdFit 광고 단위 통합:**
    *   **상단 리더보드 광고 (홈페이지 전용):**
        *   **목표:** 홈페이지 최상단에 눈에 띄는 리더보드 광고를 배치하여 클릭률 및 수익 극대화.
        *   **위치:** `index.html`의 `.content-wrapper` 최상단 (헤더 위).
        *   **디자인:** 사이트 테마와 어울리도록 중앙 정렬 및 적절한 여백 제공 (`.top-ad-container`).
        *   **광고 코드:**
            ```html
            <ins class="kakao_ad_area" style="display:none;"
              data-ad-unit = "DAN-EVlth33UH8CTDu2u"
              data-ad-width = "728"
              data-ad-height = "90"></ins>
            <script type="text/javascript" src="//t1.daumcdn.net/kas/static/ba.min.js" async></script>
            ```
    *   **하단 사이드바/피드 광고 (전체 페이지):**
        *   **목표:** 사이트 우측에 고정 광고 단위를 배치하여 수익 보완 및 가독성 유지.
        *   **적용 범위:** `404.html`을 제외한 모든 `.html` 파일.
        *   **구현:** 각 파일의 `</body>` 태그 바로 앞에 `.fixed-side-ad` 컨테이너로 감싼 Kakao AdFit 광고 코드 삽입.
        *   **특이사항 (홈페이지):** `index.html`에서는 쿠팡 파트너스 배너가 해당 사이드바 상단에 함께 배치됨.
        *   **디자인:** 데스크탑(1400px 이상)에서 우측 중앙에 고정 배치되며, 모바일 또는 화면 높이가 낮은 환경(850px 미만)에서는 숨김 처리.
        *   **광고 코드:**
            ```html
            <!-- 우측 고정 사이드바 광고 -->
            <div class="fixed-side-ad">
              <ins class="kakao_ad_area" style="display:none;"
                data-ad-unit = "DAN-KMCr4AoIjIDsi9XA"
                data-ad-width = "320"
                data-ad-height = "480"></ins>
            ```
